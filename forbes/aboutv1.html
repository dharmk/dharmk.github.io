About this visualisation

Structure of the Narrative Visualization

The narrative visualization presented here uses the Martini glass structure. It starts with a basic introduction of the area, some insights about the data and then allows the user to further explore the dataset on one's own on the last dashboard.

Scenes

As mentioned above the visualization contains five scenes. The first scens provides a high level visualization of the data, second scene presents a parallel coordinate chart to allow multi-dimensional visualization easy. Third and fourth scenes use a force directed layout to provide more insights in the variance for profits overall and across each sector. The final scene provides a dashboard for users with three linked charts to explore the data on one's own. I have also added some slides in the beginning to set the context and provide information about the data and problem domain as well as there is also a summary slide added in the end.

Annotations

Specific annotations are used in some charts to provide contextual details. Tooltips are also added in all the charts. The colors for the tooltips are annotations are chosen so as they appear soft on eyes. Consistency is maintained across all the charts by abstracting both the tooltip and annotation classes.

Parameters

The last scens is an interactive dashboard with three charts. The top chart is a bar chart which shows average profits across each sector. The other two charts show the profit for companies in the selected sector, which is used as a parameter here. The selection of the paramter is implicit, one needs to click on any sector's bar to activate it.

Triggers

The interactive dashboard uses sector as a parameter. The selection of sector is used as a trigger to update the other two linked charts which shows the top and bottom 20 companies in the selected sector with respect to the profit rates.

Design Choices

Colors for the charts are selected using ColorBrewer. Specific care has been taken to select correct colors (ordinal/quantitative). The canvas used for all the charts use soft colors for background information. It also provides dotted lines for minor values to make it easier to read the data. For comparing the profits across sectors note that I have used a force directed layout and not a sorted bar chart. This was done mostly because the no of data points were large and FDL provided a simpler more visually appealing layout which serves the purpose.
