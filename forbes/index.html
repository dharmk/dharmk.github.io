What is Forbes 2000?

The Forbes Global 2000 is an annual ranking of the top 2,000 public companies in the world by Forbes magazine.


These companies are ranked by various metrics like its Assets, Market Value, Sales and Profits.


The Forbes Global 2000 is a useful indicator of which are the leading public companies in the world, but it is only an interpretation, as only public companies are listed. The results are not definitive; any change to the criteria would produce a different list.


Reference : Forbes 2000: Methodology


Data Source: Data has been downloaded from Kaggle's public datasets section.


This contains the Forbes 2000 companies for 3 years viz 2013-2015-2017.
The attributes in the dataset are:

Rank				- Rank of the company for that year
Company				- Name of the company
Country				- Country it belongs to
Sales				- Sales that corresponding year (in billion $)
Market Value			- Market value (in billion $)
Profit				- Profit earned that year (in %)
Assets				- Total revenue the company has (in billion $)
Sector				- Expertise of the company like Finance or Technology,etc.

For this project I am using only one year data.

What am I trying to do?

I am using a subset of this data to gain insights about the top 500 companies using the concepts learned as part of CS 498 - Data Visualization course from University of Illinois, Urbana-Champaign. As part of this data-visualization project I am doing the following:


1. Select the top 500 companies. We will call if Forbes 500.


2. Analyse the data to gain some insights about the ranking metrics.


3. Gain insights about the Profit margins across various sectors and variances.


4. Allow the user to explore the profit margins for companies across various sectors.


A Martini Glass structure has been used for this final project. It starts with a basic introduction of the area, some insights about the data and then allows the user to further explore the dataset on one's own.


Everything has been done from scratch using D3 library, including all chart types, annotations, tooltips and interactivity.


Utility classes have been created for each element of visualization for consistency across the entire project.


However, since the project is slightly big, some legends, tooltips and annotations are missing from few places. I am in the process of updating all the charts but representative usage are present for each during the course of entire project. Also interactivity has been added only on the last dashboard which is representative of using a parameter and triggers.
